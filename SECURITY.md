# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities **privately** — do not open a public
issue. Use either channel:

- GitHub: **Security → Advisories → Report a vulnerability** on this repository
- Email: kneuralabs@gmail.com

Include steps to reproduce and the affected URL/file. We aim to acknowledge
reports within 72 hours.

## Scope

This repository is part of the KneuraOne environment (`kneuralabs/*`).
Reports for any KneuraOne repository are welcome through the same channels.
